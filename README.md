## League++
A league client written in c++

![](./media/preview.png)

## features

### Champion select controller
* Auto pick & ban champs

### Profile controller
* Set rank
* Set mastery
* Clear tokens

### Lobby controller
* Auto accept match
* Dodge match
* Auto dodge when on unwanted lane
